# Settlements Mod

A Fabric mod for Minecraft 1.20.1 that enhances villages with town management features through interactive lecterns.

## Features

- Settlement management system
- Building construction and material management
- Villager tracking and interaction
- Interactive lectern-based UI

## Setup

For setup instructions please see the [fabric documentation page](https://docs.fabricmc.net/develop/getting-started/setting-up) that relates to the IDE that you are using.

## Building

```bash
./gradlew build
```

The built mod JAR will be in `build/libs/`.
